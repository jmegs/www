This site lovingly built by hand with the [Astro](https://astro.build) framework — much to the chagrin of my employer, a WYSIWYG website builder. The type is TASA Orbiter by [Local Remote](https://www.localremote.co/) for the [Taiwan Space Agency](https://www.tasa.org.tw/index.php?ln=en) with [Phosphor Icons](https://phosphoricons.com/) by Helena Zhang and Tobias Fried. It is hosted on [Vercel](https://vercel.com) and the source is [available on Github](https://github.com/jmegs/www).
